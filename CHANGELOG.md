# Changelog

All notable changes to this static documentation portal are recorded here.

## Unreleased

- Initial GitHub-ready static portal for the TAFFISH RNA-seq flow family.
- Added flow-family overview, quickstart, input-preparation guide, FAQ, report
  interpretation guide, and per-flow manuals.
- Added curated yeast SNF2 example report.
- Added the de novo RNA-seq route map, manuals, quickstart guidance, FAQ
  entries, and a 24-sample yeast de novo standard-flow example report.
- Added repository health files, license files, issue templates, pull request
  template, Pages deployment workflow, and static link checker.
- Refreshed the portal for the RNA-seq 0.3 flow family: subflows
  `0.2.0-r1`, `rnaseq-standard-flow 0.3.0-r1`, and
  `rnaseq-report-flow 0.3.0-r2`.
- Replaced the reference and de novo yeast example reports with current
  standalone report-flow outputs, including embedded tables, plots, and QC HTML
  payloads.
