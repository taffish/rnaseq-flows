Reporting报告生成

rnaseq-report-flow

This flow is a static collection and interpretation layer. It gathers upstream outputs, plots, tables, linked QC HTML, commands, versions, methods, and manifests into a bilingual report and a longer interpretation guide.

这个流程是静态收集和解读层。它把上游输出、图、表、QC HTML 链接、命令、版本、方法和 manifest 汇总成双语报告和长文解读指南。

0.1.0-r4Final delivery layer最终交付层GitHub

Collect a standard-flow run

收集标准流程结果

taf-rnaseq-report-flow \
  --standard-out rnaseq-standard-out \
  --project-name "Yeast SNF2 RNA-seq" \
  --outdir report-out

Collect separate subflows

收集独立子流程

taf-rnaseq-report-flow \
  --reference-out ref-out \
  --expression-out expression-out \
  --de-out de-out \
  --enrichment-out enrichment-out \
  --outdir report-out

Input requirements

输入要求

Pass either --standard-out or one or more upstream flow output directories. The report flow reads existing outputs; it does not rerun analysis. A completed standard-flow directory is easiest, but manually run subflows can also be assembled if their output directories keep the TAFFISH RNA-seq layout.

可以传入 --standard-out，也可以传入一个或多个上游流程输出目录。报告流程只读取已有结果，不重新运行分析。完整 standard-flow 目录最简单；如果是手动运行的多个子流程，只要它们保留 TAFFISH RNA-seq 标准输出结构，也可以拼装报告。

Standard collection

收集总流程结果

--standard-out rnaseq-standard-out auto-discovers reference, expression, alignment, count, alignment QC, DE, enrichment, plot collections, and report inputs under the standard directory tree.

--standard-out rnaseq-standard-out 会自动发现标准目录树中的 reference、expression、alignment、count、alignment QC、DE、enrichment、plots 和其他报告输入。

Subflow collection

收集独立子流程

Pass --reference-out, --expression-out, --de-out, --enrichment-out, and optional alignment/count/QC directories when the subflows were run separately.

当各子流程是单独运行时，传入 --reference-out、--expression-out、--de-out、--enrichment-out，以及可选的 alignment/count/QC 目录。

How it connects

如何接上下游

It is the final step in standard-flow and can also be run later to regenerate a report from existing outputs. It should not be used to modify upstream results; upstream analysis remains the source of truth.

它是标准流程的最后一步，也可以稍后基于已有输出重新生成报告。它不应修改上游结果；上游分析目录仍然是结果来源。

Key outputs and limits

关键输出与边界

Primary outputs are 04_reports/rnaseq_report.html, 04_reports/report_interpretation.html, plot/table indexes, copied or linked HTML reports, command/version summaries, and provenance records. The flow does not rerun DE, enrichment, QC, or quantification.

主要输出是 04_reports/rnaseq_report.html、04_reports/report_interpretation.html、图表索引、复制或链接的 HTML 子报告、命令/版本摘要和溯源记录。这个流程不会重新运行 DE、富集、QC 或表达定量。
