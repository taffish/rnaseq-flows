Quickstart快速开始

Run the standard RNA-seq route

运行标准 RNA-seq 路线

Prepare local FASTQ files, a genome FASTA, a matching GTF/GFF3 annotation, sample metadata, and an offline GMT gene-set file. The flow does not download reference resources during normal execution.

准备本地 FASTQ、基因组 FASTA、匹配的 GTF/GFF3 注释、样本元数据和离线 GMT 基因集文件。流程正常运行时不会联网下载参考资源。

Minimal command

最小命令

taf-rnaseq-standard-flow \
  --samples samples.tsv \
  --genome genome.fa \
  --annotation genes.gff3 \
  --metadata metadata.tsv \
  --design '~ condition' \
  --contrast condition:treated:control \
  --gene-sets gene_sets.gmt \
  --background background.tsv \
  --outdir rnaseq-standard-out

Route choices

路线选择

The default route uses Salmon/tximport counts for differential expression. Add --route both to generate alignment/count/QC evidence. Add --route both --de-source featurecounts when you want DESeq2 to use featureCounts instead of Salmon counts.

默认路线使用 Salmon/tximport 计数做差异表达。添加 --route both 可以生成比对、计数和 QC 证据；如果希望 DESeq2 使用 featureCounts 而不是 Salmon 计数，使用 --route both --de-source featurecounts。
