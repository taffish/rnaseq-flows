FAQ常见问题

Common boundary questions

常见边界问题

Can I run only expression quantification?

只有表达矩阵需求可以运行吗？

Yes. Use the expression flow or run the standard flow until the expression outputs are available. Differential expression requires meaningful experimental groups.

可以。可以单独使用表达定量流程，或运行 rnaseq-standard-flow 并使用其中的表达矩阵输出。差异表达需要有意义的实验分组。

Does the flow download databases?

流程会下载数据库吗？

No. Genome, annotation, gene sets, and background resources are explicit inputs.

不会。基因组、注释、基因集和背景基因列表都是显式输入。

Is this a Nextflow replacement?

这是 Nextflow 替代品吗？

No. TAFFISH standardizes versioned command units and shell-native routes. Workflow engines can still orchestrate larger production DAGs.

不是。TAFFISH 标准化可版本化命令单元和 shell-native 路线。大型生产 DAG 仍然可以由工作流引擎编排。

Why separate report-flow?

为什么报告单独做成一个流程？

Reporting is a static collection and interpretation layer. It should aggregate upstream outputs rather than rerun analysis.

报告层负责静态收集和解释，应该聚合上游输出，而不是重新运行分析。
