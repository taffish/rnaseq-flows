How to read the final report

如何阅读最终报告

Start with sample and QC sections before interpreting biological signal. Low-quality reads, inconsistent mapping, or outlier samples can change every downstream conclusion.

解读生物学信号前，先看样本和 QC 部分。低质量 reads、不一致的比对情况或异常样本都会影响后续结论。

Expression matrices

表达矩阵

Gene counts are used for statistical testing; TPM-like summaries are useful for inspection and expression-level communication.

gene counts 用于统计检验；TPM 类结果更适合人工检查和表达水平沟通。

Differential expression

差异表达

Read PCA and correlation plots first, then MA/volcano plots, then gene tables and heatmaps. Effect size and adjusted P value answer different questions.

先看 PCA 和相关性图，再看 MA/volcano 图，最后看 gene table 和 heatmap。效应量和校正 P 值回答的是不同问题。

Enrichment

功能富集

ORA summarizes significant-gene lists, while GSEA reads the full ranked list. Treat enrichment as a biological hypothesis generator, not as proof that a pathway is active.

ORA 汇总显著基因列表，GSEA 读取完整排序基因列表。富集结果更像生物学假设生成器，不应直接等同于通路被证明激活。

Provenance溯源

Use commands, versions, logs, and manifests to determine exactly how the report was produced.

通过 commands、versions、logs 和 manifest 可以确认报告到底如何生成。
