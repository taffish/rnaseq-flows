rnaseq-standard-flow 0.2.0-r2 runs the RNA-seq standard route by composing published TAFFISH RNA-seq subflows.

The default --mode reference route is compatible with r2: it prepares a reference bundle, quantifies transcript expression with Salmon/tximport, runs DESeq2 differential expression, runs offline GMT enrichment, and renders a static project report.
With --route both in reference mode it also runs HISAT2 alignment, alignment-level QC, and featureCounts gene counting. Differential expression uses --de-source to choose Salmon/tximport or featureCounts counts.
The explicit --mode denovo route replaces reference indexing and reference-guided quantification with de novo transcriptome assembly, transcript-level Salmon quantification, and homology-derived annotation/gene sets before DE/enrichment/report.
The flow does not download references, gene sets, or databases at runtime. It writes all generated files under the explicit output directory.
