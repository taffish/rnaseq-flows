# Support

For help with the TAFFISH RNA-seq flow family, please choose the smallest public
example that reproduces the question.

## Good Issue Reports Include

- The flow name and version, for example `taf-rnaseq-standard-flow 0.2.0-r2`.
- The analysis mode, for example reference mode or `--mode denovo`.
- The exact command, with private paths or sample names anonymized if needed.
- A short description of the input table shape.
- The relevant log excerpt.
- Whether the issue is about documentation, report interpretation, installation,
  container execution, or biological design.

## Where to Ask

- Documentation site issues: this repository.
- A specific flow command failure: the matching `rnaseq-*-flow` repository.
- TAFFISH runtime, install, backend, or container behavior: the core `taffish`
  repository.
- Hub/index discovery problems: the TAFFISH Hub or index repository.

## Private Data

Please do not upload private biological data to public GitHub issues. Use public
accessions, synthetic fixtures, or redacted logs whenever possible.
